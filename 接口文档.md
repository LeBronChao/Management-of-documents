

# 登录

### path：  /User/Login      method:POST

## 参数：

| 参数名    | 参数说明 | 参数类型 | 备注 |
| --------- | -------- | -------- | ---- |
| user_name | 用户名   | string   |      |
| password  | 密码     | string   |      |



## 响应参数

| 参数名    | 参数说明     | 参数类型 | 备注                           |
| --------- | ------------ | -------- | ------------------------------ |
| user_name | 用户名       | string   | 通过用户名查找，用户名不可变更 |
| reg_date  | 注册日期     | string   |                                |
| name      | 姓名         | string   |                                |
| phone     | 手机号       | string   |                                |
| job_no    | 工号         | string   |                                |
| unit      | 单位         | string   |                                |
| jur       | 权限         | int      |                                |
| status    | 是否登陆成功 | bool     |                                |
| errmsg    | 错误信息     | string   |                                |
