

# 登录

### path：  /Login      method:POST

## 参数：

| 参数名    | 参数说明 | 参数类型 | 备注 |
| --------- | -------- | -------- | ---- |
| user_name | 用户名   | string   |      |
| password  | 密码     | string   |      |



## 返回值

| 参数名 | 参数说明     | 参数类型 | 备注 |
| ------ | ------------ | -------- | ---- |
| status | 是否成功登录 | bool     |      |
| errmsg | 错误信息     | string   |      |





# 注册

### path： /Register    method：POST

### 备注：注册的用户权限默认为普通用户

## 参数：

| 参数名    | 参数说明 | 参数类型 | 备注 |
| --------- | -------- | -------- | ---- |
| user_name | 用户名   | string   |      |
| password  | 密码     | string   |      |
| name      | 姓名     | string   |      |
| phone     | 手机号   | string   |      |
| job_no    | 工号     | string   |      |
| unit      | 单位     | string   |      |



## 返回值

| 参数名 | 参数说明     | 参数类型 | 备注 |
| ------ | ------------ | -------- | ---- |
| status | 是否成功注册 | bool     |      |
| errmsg | 错误信息     | string   |      |
